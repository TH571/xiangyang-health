# 向阳健康 - 设计理念探索

## 设计理念选定

### 选定方案：现代健康主义 (Modern Wellness Minimalism)

**设计哲学**: 以温暖的有机形态和清晰的信息架构为核心，打造一个既专业又亲切的健康知识平台。设计强调**光线与生长**的主题，通过渐进式的视觉层级和精心选择的色彩来传达健康、希望和知识的力量。

#### 核心原则

1. **光线引导** - 使用橙色和暖色系作为主要视觉引导，象征"以光为引"的品牌理念
2. **有机流动** - 避免生硬的直线和规则网格，采用曲线和自然形态的分割线
3. **信息清晰** - 严格的排版层级和充足的空白，确保内容易读易扫
4. **温暖亲切** - 柔和的阴影、圆润的角度和温暖的色调创造信任感

#### 色彩哲学

| 颜色 | 值 | 用途 | 情感 |
|------|-----|------|------|
| 主橙色 | #FF8C42 | Logo、强调、CTA | 温暖、能量、希望 |
| 深蓝色 | #2C3E50 | 文本、背景对比 | 专业、信任、稳定 |
| 浅米色 | #F5E6D3 | 主背景、卡片 | 柔和、自然、舒适 |
| 强调红 | #E74C3C | 次级CTA | 活力、紧急性 |
| 中性灰 | #95A5A6 | 辅助文本、边框 | 平衡、中立 |

#### 布局范式

- **非对称网格**: 使用 3 列主内容 + 1 列侧边的布局，避免完全居中
- **有机分割**: 用曲线 SVG 分割线而非直线，创造流动感
- **卡片浮动**: 内容卡片采用浅阴影和微妙的悬停效果，营造深度
- **响应式堆叠**: 移动端自然堆叠，保持视觉连贯性

#### 标志元素

1. **太阳花心形 Icon** - 品牌核心，用于 Logo 和强调位置
2. **有机曲线分割线** - 在各个部分之间使用，创造流动感
3. **渐进式卡片** - 文章卡片采用浮动设计，随滚动产生视差效果
4. **温暖渐变** - 从橙色到浅米色的微妙渐变用于背景和强调区域

#### 交互哲学

- **微妙反馈**: 按钮和链接采用柔和的颜色变化而非剧烈动画
- **流畅过渡**: 所有状态变化使用 300-500ms 的缓动，创造优雅感
- **视觉引导**: 使用色彩和大小变化引导用户注意力，而非闪烁或抖动
- **悬停提升**: 卡片和按钮在悬停时产生微妙的上升和阴影增强效果

#### 动画指南

- **页面进入**: 内容从下方淡入并轻微上升（300ms cubic-bezier）
- **卡片悬停**: 阴影增强、轻微上升（200ms ease-out）
- **按钮点击**: 按钮缩小 2% 后恢复，配合背景色变化（150ms）
- **滚动触发**: 文章卡片在进入视口时产生淡入和侧向滑动（400ms）

## 实现指南

### 全局样式设置
- 在 `client/src/index.css` 中设置色彩变量和字体导入
- 使用 Tailwind 的 `@theme` 块定义自定义颜色和间距
- 在 Google Fonts 中导入思源黑体和思源宋体

### 组件设计
- 所有卡片使用 `rounded-lg` 和 `shadow-sm` 配合 `hover:shadow-md` 过渡
- 按钮采用 `bg-orange-500` 主色，`hover:bg-orange-600` 过渡
- 文本颜色使用 `text-slate-800` 作为主文本，`text-slate-600` 作为辅助

### 响应式设计
- 移动端: 单列布局，充分利用宽度
- 平板端: 两列布局，卡片并排显示
- 桌面端: 三列主内容 + 侧边栏或完整网格

### 页面结构
1. **Header**: 固定顶部导航，包含 Logo 和菜单
2. **Hero**: 全宽背景，包含品牌宣言和 CTA
3. **Sections**: 多个内容区域，各自独立的背景色和分割线
4. **Footer**: 底部信息、链接和社交媒体

## 设计决策记录

- **为什么选择现代健康主义?** 这个风格既能体现 Wix 原站点的温暖亲切感，又能提升专业度和现代感。橙色和蓝色的组合在健康领域具有高度的识别度和信任感。

- **为什么避免对称布局?** 非对称设计能创造视觉张力和动态感，使页面更有趣，同时保持信息清晰。

- **为什么使用有机曲线?** 曲线象征生长、流动和生命力，与"健康"主题高度契合，比直线更温暖。

- **为什么选择这些字体?** 思源系列字体在中文显示中具有高度的可读性和专业感，同时保持温暖的人文特质。
